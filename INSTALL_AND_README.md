## INSTALLATION

### Prerequisites
- Ubuntu 22.04 LTS or compatible
- Python 3.10+
- Git
- 16GB RAM minimum
- 100GB disk space

### Step 1: Clone Repository
```bash
git clone https://github.com/hcindus/AOS-Brain.git
cd AOS-Brain
```

### Step 2: Install Dependencies
```bash
# System dependencies
sudo apt-get update
sudo apt-get install -y python3-pip python3-venv nodejs npm

# Python dependencies
pip3 install -r requirements.txt
```

### Step 3: Configure Environment
```bash
# Create .env file
cp .env.example .env

# Edit .env with your credentials
nano .env
```

Required environment variables:
- `GITHUB_TOKEN` - For GitHub API access
- `EMAIL_PASSWORD` - For SMTP notifications
- `OPENCLAW_API_KEY` - If using OpenClaw gateway

### Step 4: Install Security Probes (Optional but Recommended)
```bash
cd security/probes
sudo bash install-probes.sh
```

### Step 5: Start Services
```bash
# Start Complete Brain v4.1
sudo systemctl start aos-brain-v4

# Start Mission Control
sudo systemctl start aos-mission-control

# Verify status
sudo systemctl status aos-brain-v4
sudo systemctl status aos-mission-control
```

### Step 6: Access Dashboard
- Mission Control: http://localhost:8080
- Brain Status API: http://localhost:8080/api/status
- Metrics (if node_exporter installed): http://localhost:9100/metrics
