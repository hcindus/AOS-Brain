# VENDOR RECRUITMENT MISSION
## Task for Dark Factory Team

**Mission:** Find and onboard compatible production vendors  
**Deadline:** 1 week  
**Goal:** 5 qualified vendors ready for integration  

---

## VENDOR CRITERIA

### Required Capabilities:
- [ ] 3D printing (FDM or SLA) OR CNC machining
- [ ] Accept file uploads (STL, OBJ, STEP, G-code)
- [ ] API-friendly or willing to integrate
- [ ] US-based (for quick turnaround)
- [ ] Accept payment per-job (not contract)

### Nice to Have:
- [ ] Existing API
- [ ] Webhook support
- [ ] Quality certifications
- [ ] Fast turnaround (24-48hr)
- [ ] Competitive pricing

---

## SEARCH STRATEGY

### 1. Online Platforms
- **Xometry.com** - Quote API available
- **Fictiv.com** - Good API, quality parts
- **Craftcloud.xyz** - 3D print aggregator
- **Hubs.com** (now Protolabs) - Network of shops
- **3D Hubs** - Local makers

### 2. Local Search
- Google Maps: "3D printing service [your city]"
- Yelp: "CNC machining"
- Makerspaces (often have commercial services)

### 3. Community Sources
- r/3Dprinting - Post service needs
- r/CNC - Machine shops
- Facebook groups: 3D Printing, CNC Machining
- Thingiverse: Contact prolific makers
