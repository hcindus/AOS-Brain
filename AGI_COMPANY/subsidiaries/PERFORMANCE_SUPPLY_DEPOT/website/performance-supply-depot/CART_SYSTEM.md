# Shopping Cart System with Stripe Integration
## AGI Company E-Commerce Implementation

### Files Created:
1. cart.js - Shopping cart functionality
2. cart.html - Cart page
3. products.html - Product catalog with add-to-cart
4. stripe_checkout.js - Real Stripe integration
5. webhook_handler.py - Payment confirmation

### Features:
- Add multiple products to cart
- Update quantities
- Remove items
- Calculate totals
- Real Stripe payment
- Webhook confirmation
- Email receipts

### Payment Methods:
- Credit/Debit cards (Stripe)
- Crypto (optional wallet integration)

See individual files for implementation.
