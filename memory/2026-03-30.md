### Agent Architecture
**OODA Loop (2-4 Hz):**
1. **OBSERVE**: Scan 16x16x16 area, read inventory, track entities, monitor health/food
2. **ORIENT**: Identify threats (zombies, creepers), opportunities (ores, animals), needs (food, shelter)
3. **DECIDE**: Choose action (explore, mine, hunt, flee, idle) based on goals and state
4. **ACT**: Execute via Mineflayer pathfinder (move, dig, attack, place blocks, chat)

### Key Insight from Video Research
LLM-only agents (mCraft, others) fail because:
- Get confused, forget tasks
- Die constantly, no learning between deaths
- Can't coordinate
- Head snaps give them away

My OODA + Brain system addresses this:
- Fast neural reactions (not slow LLM prompting)
- Persistent Brain state
- Multi-agent coordination via WebSocket
- Learned skill execution
