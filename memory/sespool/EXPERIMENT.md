# SESPOOL EXPERIMENT
## Waste-to-Pattern Learning Study

**Started:** 2026-03-31 06:24 UTC
**Hypothesis:** Pattern recognition can emerge from unstructured, noisy, "waste" data

### The Setup
Two brains, same source data, opposite filtration:

| Miles-Brain | Mortimer-Brain |
|-------------|----------------|
| Curated, high-signal data | Raw waste, noise, redundancy |
| Clean periodic table entries | Conflicting atomic weights |
| Current slang definitions | Deprecated/Regional variants |
| Standard spellings | Archaic, alternate spellings |
| Confirmed synonym clusters | Duplicate, contradictory mappings |

### Expected Divergence
- **Miles:** Precise, structured knowledge graphs
- **Mortimer:** Emergent pattern recognition from chaos, tolerance for ambiguity

### Methodology
1. Capture waste during Miles' digestion
2. Batch and queue for Mortimer
3. Auto-feed every 5 minutes
4. Monitor Mortimer's cluster formation

### Success Metrics
- Does Mortimer form different semantic clusters?
- Does noise-filtering improve his anomaly detection?
- Can he resolve contradictions Miles cannot?

**This is learning from the shadows.**
