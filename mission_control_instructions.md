## TROUBLESHOOTING

### Service Won't Start
```bash
# Check for errors
journalctl -u aos-mission-control -n 50

# Check if port 8080 is in use
lsof -i :8080

# Try manual start
cd /root/.openclaw/workspace/aocros/mission_control
python3 server.py
```

### Page Won't Load
- Verify service is running: `systemctl is-active aos-mission-control`
- Check firewall: `ufw status`
- Try direct IP: http://127.0.0.1:8080

### Visual Issues
- Clear browser cache
- Try different browser
- Check WebGL support: https://get.webgl.org/
